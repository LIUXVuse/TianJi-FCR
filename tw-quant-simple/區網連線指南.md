# 🌐 區網連線設定指南

## ✅ 快速啟動（整合版）

目前天機·火控雷達已整合量化回測系統，只需一個指令即可同時啟動兩個服務：

```bash
# 進入專案根目錄
cd 天機·火控雷達-(tianji-fcr)

# 一鍵啟動（前端 + 量化後端）
npm run dev
```

啟動後會看到：

- `前端` (藍色) - Vite 開發伺服器，port 3000
- `量化` (綠色) - FastAPI 後端，port 8000

---

## 🔗 連線方式

| 裝置 | 連線網址 |
|------|----------|
| 本機 | `http://localhost:3000` |
| 區網其他電腦 | `http://[你的IP]:3000` |
| 手機（同 WiFi） | `http://[你的IP]:3000` |

---

## 📝 查詢你的區網 IP

```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

輸出範例：

```
inet 192.168.x.x netmask 0xffffff00 broadcast 192.168.x.255
```

---

## ⚠️ 安全提醒

- 服務預設監聽 `0.0.0.0`（區網可連）
- 在公共 WiFi 環境請特別注意安全性
- 家用或公司內網使用是安全的

---

## 🔧 常見問題

### Q: 其他電腦連不上？

檢查以下項目：

1. 兩台電腦是否在同一個 WiFi / 區網
2. Mac 防火牆設定（系統設定 → 網路 → 防火牆）
3. 確認服務有正確啟動

### Q: 如何只啟動單一服務？

```bash
# 只啟動前端
npm run dev:frontend

# 只啟動量化後端
npm run dev:quant
```
